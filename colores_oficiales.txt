1. PALETA DE COLORES OFICIAL (Maestro Éxodo AI)
================================================================================

1. NEGRO CÁLIDO ÉXODO (Fondo Principal)
   - Hexadecimal: #0E0C0A
   - Código Flutter: Color(0xFF0E0C0A)
   - Uso: Fondo general de la aplicación en Modo Oscuro.

2. SUPERFICIE TARJETAS / TABS / COMPOSER
   - Hexadecimal: #1A1612
   - Código Flutter: Color(0xFF1A1612)
   - Uso: Tarjetas de chat, pestañas superiores, campos de entrada.

3. BORDES SUTILES
   - Hexadecimal: #2A241D
   - Código Flutter: Color(0xFF2A241D)
   - Uso: Divisores y contornos de tarjetas.

4. ÁMBAR ÉXODO (Primario / Resplandores)
   - Hexadecimal: #C9933A
   - Código Flutter: Color(0xFFC9933A)
   - Resplandor traslúcido (20% alfa): Color(0x33C9933A)
   - Uso: Botones activos, acentos, íconos principales, resplandor ambiental.

5. BLANCO HUESO / YESO (Texto Principal / Botón Enviar)
   - Hexadecimal: #F5F2EB
   - Código Flutter: Color(0xFFF5F2EB)
   - Uso: Títulos principales, texto destacado, botón de enviar en Modo Oscuro.

6. GRIS ARENA (Texto Secundario)
   - Hexadecimal: #9E9689
   - Código Flutter: Color(0xFF9E9689)
   - Uso: Subtítulos, horas, conteo de tokens, placeholders.

7. ROJO ÉXODO (Alertas / Errores)
   - Hexadecimal: #E5534B
   - Código Flutter: Color(0xFFE5534B)
   - Uso: Mensajes de error, desconexión de base de datos.
